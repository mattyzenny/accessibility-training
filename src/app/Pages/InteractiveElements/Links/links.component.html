Links are simple components that are often misued. Buttons and links may sometimes look similar, but they should perform different actions. Links are expected to navigate to a different page or resource, while buttons are used to perform an action. In the same manner, buttons and links should not be styled to look like one another. If you remember links go somewhere, and buttons do something, then you should be well equipped to utilize the two components correctly.

Required

  • A link's purpose must be clear from its text or surrounding context. Click here! is not an appropriate label for a link. A user should be able to navigate to the link and understand it's purpose.
  • A link must have a programmatically discernable text, and must be semantically identifiable as a link. This means it is marked up as a link <a> and has visible or accessible text that match.
  • It is required that all elements (not just links) are 24x24 pixels wide, or have a 24x24 margin between the next interactive element.
  • A link must be keyboard navigable.
  • Color cannot be the only visual means of conveying information. If a link changes color on hover, then it must also have an underline or other visual indicator. It is best practice to style links with an underline by default, and to have in-line links contrast to their surrounding text.
  • When a link is truncated, it must either display the full text on focus and on hover, or in the linked page iteself.

Best Practice

  • A link should be styled with an underline by default, and should have a contrastive color to its surrounding texts.
  • A link should explicitly state where it will navigate to. For example, a common trend is for a link that opens a new tab should say "Opens in a new window".
  • Do not add the word 'link' to the end of a link. It is redundant and unnecessary.

Links in a paragraph must be distinguishable from the surrounding text. The example uses a different font family, underlines the text, and uses color. At least 2 of these are required for a link to be considered accessible.
