Welcome to the Angular Accessibility Training. This application is designed to showcase some of the accessibility features that are baked into our Cel-Components. The purpose is to introduce accessibility in a practical and developer centric manner.

Before you begin, you will need to install a screen reader, or Assistive Technology (AT) for short.

Windows devices use NVDA.

Mac devices use Voiceover.

You can use the confluence page Assistive Technology or Deque's Screen Reader Keyboard Shortcuts that can help you navigate the application.
