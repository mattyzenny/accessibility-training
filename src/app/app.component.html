Buttons

Buttons are an important part of Design. They are used in just about every page within Realize. By inheriting a cel-Button, you will want to make sure that all buttons are accessible. Below are 2 different buttons. The first button has a label (data-label) attribute that is used to set the visible label. The second button has a data-button-aria-label attribute that is used to override the aria-label attribute.

These buttons look identical. However when screen reader is used, they are completely different.

Input Validation

Input field validation is important for accessibility. Without the correct semantic association, a screen reader may not know which input the error is referring to or even worse, they may not be aware of an error at all.
