Radio group buttons are another important element within a form. They allow users to select one option from a list of options. Radio buttons are often used in groups, where only one option can be selected at a time. Each radio button requires a label for the button, and some sort of group label to help the user understand the context.

How to make a radio group accessible?

  • A button must have a visible label.
  • If a button has an accessible label, it must start with the visible text.
  • The name, role, state, & value must be programmatically determinable. (role is often baked into the native HTML <button> component | name is the visible text | value, state are required for more complex functionality)
